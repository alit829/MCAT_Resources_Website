About

I made this website so that I can help fellow MCAT & DAT test takers by providing a library that contains resources that have been valued by other test takers based on their responses on Reddit and their score. When I first took the MCAT, I didn't know what resources would be helpful to have, and that wasted precious time that could've been spent studying or doing something useful.

As well, I wanted to learn coding, so I thought why not make a website rather just post an Excel file containing the different resources onto Reddit. I hope I have been some help to you fellow test takers.

If you think I missed a valuable resource that should be added, I have left a link to my Reddit post where you can leave a comment with the resource's name and link so that I can add it to the list.

Enjoy :)
